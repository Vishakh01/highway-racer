## 📖 Overview

Highway Racer is a feature-rich arcade racing game developed entirely in **Python** using **Pygame**.

The game combines smooth vehicle controls, realistic driving physics, intelligent traffic AI, dynamic weather effects, and a polished user interface to deliver an enjoyable highway driving experience.

Whether you're learning game development or looking for a complete Pygame project, this repository demonstrates clean object-oriented programming, game loops, physics simulation, sprite management, and UI design.

# 🎮 Controls

| Key | Action |
|------|--------|
| **W / Up Arrow** | Accelerate |
| **S / Down Arrow** | Brake / Reverse |
| **A / Left Arrow** | Turn Left |
| **D / Right Arrow** | Turn Right |
| **Space** | Hand Brake / Drift |
| **Left Shift** | Nitro Boost |
| **Esc** | Pause Game |
| **Enter** | Start / Restart |

# ⚙ Installation

Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/highway-racer-pygame.git
```

Move into the project.

```bash
cd highway-racer-pygame
```

Install dependencies.

```bash
pip install pygame
```

Run the game.

```bash
python main.py
```
